# 🎨 Prompt 分类编号系统（PCC：Prompt Classification Code）

一个面向多作者、多风格、多用途的图像创意资料库编号系统，参考图书馆分类法（LCC），支持无限扩展与多来源内容采集。

---

## 📦 编号结构格式

```
<主类代码>-<风格代码>-<用途缩写>-<流水号>
``

### 字段说明

- **主类代码**：参考LCC主类（如 N、T、Q 等）
- **细分类代码**：LCC二级/三级细分（如 NC、TR、QA 等，若无可省略）
- **风格代码**：自定义风格（如 3DHE、PXL、VOXL 等，可扩展）
- **用途缩写**：内容用途（如 SNS、COM、MEM 等）
- **作者缩写**：内容采集/整理者（如 HWS、ZZZ、EGB 等，便于多作者管理）
- **年份**：内容采集或创作年份（如 2024）
- **流水号**：同一分类下的唯一编号，三位数起（如 001）

### 示例
`
```
N-NC-2DIL-SNS-HWS-2024-001
```
表示：艺术 > 插画 > 2D插画风格 > 社交媒体用途 > 作者HWS > 2024年 > 第1条
